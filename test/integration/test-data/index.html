ExternalSource Test API

This API provides test endpoints for ExternalSource integration testing.

Available Endpoints

GET /health

Health check endpoint

GET /api/v1/config

Basic application configuration

GET /api/v1/users

User data for testing

GET /api/v1/settings

Application settings

GET /api/v1/secure-config

Authenticated endpoint (requires Authorization: Bearer test-token-123)

GET /api/v1/environment-config?env={dev|staging|production}

Environment-specific configuration

GET /api/v1/slow-config

Slow endpoint (2 second delay) for timeout testing

GET /api/v1/flaky-config

Flaky endpoint (30% error rate) for retry testing
